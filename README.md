# Conomy

### Тестовое задание для conomy.ru

### Запуск
* Создать виртуальную среду и установить зависимости
    * ```python -m venv venv```
    * ```source venv/scripts/activate```
    * ```pip install -r requirements.txt```
* Применить миграции и запустить сервер
    * ```./manage.py migrate```
    * ```./manage.py runserver```
